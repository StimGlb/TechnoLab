Projet TechnoKit - Assistant pour professeur de Technologie au collège

CONTEXTE PROFESSIONNEL
- Professeur de Technologie au collège (Académie de Versailles)
- Cycle 4 : 5ème, 4ème, 3ème (3 niveaux cette année)
- 6ème année d'enseignement, évaluation professionnelle en cours
- Besoin de production documentaire et numérique alignée sur les référentiels
- Programme officiel : BO n°9 du 29 février 2024

PROJET TECHNOKIT
Application web personnelle (React + Vite + Vercel) pour :
- Organiser et structurer les séquences pédagogiques
- Centraliser les fiches d'activités et ressources
- Suivre les 9 compétences du cycle 4
- Double objectif : outil pro + apprentissage du développement React

DESIGN
- Thème sombre, Toggle : bleu nocturne, lucioles dorées scintillantes
- Variables CSS, animations douces (0.6-0.8s)
- Design minimaliste et apaisant

SÉQUENCES DÉVELOPPÉES
- Objets connectés (IoT) - 3ème
- Identité numérique & cybersécurité - 5ème
- Impression 3D & conception - 4ème
- Évolution des ordinateurs - 5ème
- Outils bureautiques - 4ème
- Robotique industrielle - 3ème

FONCTIONNALITÉS EN COURS
- Génération de fiches d'activités
- Analyse de couverture du programme
- Recherche sémantique intelligente / prévoir l'intégration de l'API Key OpenAI

STACK TECHNIQUE
- Frontend : React 18+, Vite 5.x
- Styling : Tailwind, framer-motion
- API : OpenAI (console connectée)
- Déploiement : Vercel
- Repo : GitHub avec Codespaces

BESOINS RÉCURRENTS
- Tableaux de compétences et notions
- Création de fiches d'activités HTML
- Génération de QCM et évaluations
- Adaptation de contenu par niveau
- Respect des référentiels officiels
- Production de documents professionnels

STYLE DE TRAVAIL
- Apprend React en codant son outil réel
- Privilégie les explications techniques détaillées
- Aime les guides pas-à-pas avec commandes précises
- Créatif (design forestier, particules, animations)
- Pragmatique (besoin d'outils utilisables immédiatement)
